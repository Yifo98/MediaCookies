## Global rules

- Two depth anchors in every beat: left-side editorial message and right-side real product surface.
- Persistent background texture: violet/cyan radial light, a large low-opacity production mark, two edge-anchored boundary rules, chapter/time metadata, and a subtle grain field.
- Primary transition: 0.5-second blur crossfade. Use a vertical push only at the popup-to-workbench expansion.
- Every beat has a visible subtitle line. No narration is needed; the output still contains an AAC track.
- No real accounts, private content, Cookie values, file paths, or notification chrome.
- Use the actual screenshots and logo assets already adopted under `assets/`.
